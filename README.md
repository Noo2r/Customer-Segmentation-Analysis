# Customer Segmentation Analysis

## Overview
This project focuses on analyzing wholesale customer data and building customer segmentation models using unsupervised machine learning techniques.

The project includes data preprocessing, exploratory data analysis (EDA), dimensionality reduction using PCA, clustering, and data visualization.

---

## Objectives
- Analyze customer purchasing behavior
- Segment customers into meaningful groups
- Reduce feature complexity using PCA
- Visualize customer clusters and business insights
- Support data-driven marketing and decision-making

---

## Dataset
The dataset contains wholesale customer spending data collected from clients of a wholesale distributor in Lisbon, Portugal.

### Features Include:
- Fresh Products
- Milk Products
- Grocery Products
- Frozen Products
- Detergents & Paper
- Delicatessen Products
- Customer Channel
- Region
